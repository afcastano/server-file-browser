server-file-browser
===================

Browse server files.

Functionalities:
Move files
Delete files
Create dir
Upload subtitles

Pending:
Rename files

Pending:
Deploy instructions.

Run `npm install -g grunt-cli`

run `grunt build --force`

Server side tests


Development:
Run `grunt update` to update sources

Check npm scripts to run the server

Using https://github.com/logicbomb/lvlFileUpload for uploading subtitles.
